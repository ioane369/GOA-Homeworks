Explaining the CSS priority tree

The CSS priority tree determines which styles are applied to an element when multiple rules are present. The specificity is calculated based on the types of selectors used.

Specificity calculation

  1. Inline Styles

    Inline styles have the highest priority. Styles written directly in the HTML element using the style attribute will override all other styles. For example: div style="color: red;".

  2. IDs

    Selectors with IDs have the next highest priority. For example: #header { color: blue; } has higher specificity than a class or a tag selector.

  3. Classes

    Class selectors are next in line. For example: .button { color: green; }.

  4. Elements

    Element selectors have the lowest specificity. For example: p { color: black; }.
